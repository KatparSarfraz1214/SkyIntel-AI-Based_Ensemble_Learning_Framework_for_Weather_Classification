#  SkyIntel: AI-Based Ensemble Learning Framework for Weather Classification

## Overview

**SkyIntel** is an intelligent weather classification framework that leverages the power of **Ensemble Learning** to accurately classify weather conditions based on meteorological data. The system integrates machine learning techniques to analyze weather attributes such as precipitation, temperature, wind speed, and temporal features, providing reliable weather condition predictions.

The project employs a **Random Forest Classifier**, a robust ensemble learning algorithm, to improve prediction accuracy and model generalization. Additionally, SkyIntel features an interactive **Streamlit-based web application** that enables users to perform real-time weather classification through a modern and user-friendly interface.
